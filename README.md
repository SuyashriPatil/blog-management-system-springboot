# 📘 Blog Management System

This is a full-stack web application developed using Spring Boot and Thymeleaf that allows users to create, view, update, and delete blog posts.

## 🚀 Features

- 📝 Create new blog posts
- 📄 View all blog posts
- ✏️ Edit existing posts
- 🗑️ Delete posts using ID
- 🎨 Attractive UI using HTML & CSS
- 🔗 Dynamic data rendering using Thymeleaf

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- HTML, CSS

## 📂 Project Structure

- Controller Layer → Handles HTTP requests
- Model Layer → Represents Post entity
- Repository Layer → Handles database operations
- Templates → Thymeleaf HTML pages

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/your-username/blog-management-system-springboot.git

2. Open in IntelliJ / Eclipse

3. Configure MySQL database:
   - Create database: blogpost
   - Update username & password in application.properties

4. Run the application

5. Open browser:
   http://localhost:8080/
